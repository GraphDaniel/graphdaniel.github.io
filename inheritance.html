Специфичность

Специфичность — это алгоритм, благодаря которому браузер определяет, какие именно стили из всего набора применить к элементу. У каждого селектора есть свой вес. Если у одного элемента сразу несколько селекторов, то браузер применяет стили селектора с самым большим весом.

Типы селекторов по убыванию специфичности:

  1. Селекторы по ID.
  2. Селекторы по классу, атрибуту и с псевдоклассами.
  3. Селекторы по тегу, селекторы с псевдоэлементами.

Также, существует ключевое слово !important, которое насильно применяет свойство, после которого написано. Его стоит использовать очень осторожно и не злоупотреблять им.

Каскадность

Каскад определяет выбор CSS-правил, применяемых в случае влияния на стиль элемента нескольких объектов.

Три фактора в порядке возрастания:

  • Порядок следования(побеждает правило, которое идёт последним в CSS).
  • Специфичность
  • !important

Наследование

Наследование - это способность элементов-потомков перенимать правила форматирования, которые присвоены их предкам.

Элементы могут быть как наследуемыми, так и ненаследуемыми.

За контроль наследования отвечают несколько значений:

  • inherit - элемент наследует значение свойства у своего предка.
  • initial - значение свойства по умолчанию. Значение может отличаться от браузерных стилей.
  • unset - для наследуемых свойств будет равен inherit, initial для ненаследуемых. Используется только для свойства all.
  • revert - применяет стили браузера по умолчанию.
