Таблицы в HTML

Создание таблиц

Любая таблица заключается в теги <table></table>.

Строки таблицы создаются с помощью <tr>(tr - table row).

Ячейки таблицы с помощью <td>(td - table data).

За заголовки таблицы отвечает <th>.

Атрибуты colspan и rowspan отвечают за распространение ячеек на несколько столбцов или строк соответсвенно.

Стилизацию для столбцов можно удобно задать атрибутами <colgroup> и <col>

Для более удобной стилизации и понятной вёрстки можно использовать <thead>, <tbody>, <tfoot>.
