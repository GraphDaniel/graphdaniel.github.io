Конспект по изображениям в HTML

Размещение изображения на странице

Изображения размещаются с помощью одиночного тега <img>. Обязателен атрибут src. С его помощью указывается путь к изображению.

Пример: <img src="./img/monsie-pc.png">

Альтернативный текст

Атрибут alt используется для описания картинки, когда она не может быть отображена.

Пример (возьмём картинку выше): <img src="./img/monsie-pc.png" alt="Монси за ноутбуком">

Заголовок изображения

Атрибут title используется для создания всплывающей подсказки

Пример: <img src="./img/monsie-pc-title.png" alt="Монси за ноутбуком с примером всплывающей подсказки" title="Монси за ноутбуком">

Семантический контейнер и заголовок для картинки

Теги <figure> и <figcaption> используются для семантической обёртки и связывании рисунка с заголовком

Пример: 

						
							<figure>
								<img src="images/dinosaur.jpg" alt="Монси с ноутбуком">
							<figcaption>Монси с ноутбуком</figcaption>
							</figure>
